# 安装

## 本地引用

在 [GitHub 仓库](https://github.com/evenyao/GearCase)下载，或使用 git 命令克隆到本地后注册使用

```
git clone https://github.com/evenyao/GearCase.git
```

## NPM / Yarn 安装

使用 `NPM` 或者 `Yarn` 方式进行安装

```
npm i -S gear-case
```
或
```
yarn add gear-case
```

如果您已经使用了上述任意一种方式进行了安装，请继续阅览[快速上手](/get-started/README.md)章节
